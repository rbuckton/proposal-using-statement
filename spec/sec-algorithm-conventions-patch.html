Algorithm Conventions

Runtime Semantics

DisposeAndReturnIfAbrupt

Algorithm steps that say or are otherwise equivalent to:

1. DisposeAndReturnIfAbrupt(_env_, _argument_).

mean the same thing as:

1. Assert: _env_ is a declarative Environment Record or *undefined*. 1. If _argument_ is an abrupt completion, return DisposeEnvironment(_env_, _argument_). 1. If _argument_ is a Completion Record, set _argument_ to _argument_.[[Value]].

Algorithm steps that say or are otherwise equivalent to:

1. DisposeAndReturnIfAbrupt(_env_, AbstractOperation()).

mean the same thing as:

1. Assert: _env_ is a declarative Environment Record or *undefined*. 1. Let _hygenicTemp_ be AbstractOperation(). 1. If _hygenicTemp_ is an abrupt completion, return DisposeEnvironment(_env_, _hygenicTemp_). 1. If _hygenicTemp_ is a Completion Record, set _hygenicTemp_ to _hygenicTemp_.[[Value]].

Where _hygenicTemp_ is ephemeral and visible only in the steps pertaining to DisposeAndReturnIfAbrupt.

Algorithm steps that say or are otherwise equivalent to:

1. Let _result_ be AbstractOperation(DisposeAndReturnIfAbrupt(_env_, _argument_)).

mean the same thing as:

1. Assert: _env_ is a declarative Environment Record or *undefined*. 1. If _argument_ is an abrupt completion, return DisposeEnvironment(_env_, _argument_). 1. If _argument_ is a Completion Record, set _argument_ to _argument_.[[Value]]. 1. Let _result_ be AbstractOperation(_argument_).
